# mcp-carbone-hermes-mcp

Serveur MCP distant pour Hermes Carbone.

## Endpoints

- `GET /health`
- `POST /mcp`

## Deployment

- Build command: `npm install`
- Start command: `npm run start`

## Public MCP URL

Une fois déployé, l'URL MCP à connecter dans OpenAI sera :

`https://VOTRE-DOMAINE-RAILWAY/mcp`
